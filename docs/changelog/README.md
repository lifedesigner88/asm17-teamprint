# Changelog Index

## Structure
- 일자별 변경 기록은 `docs/changelog/YYYY-MM-DD.md` 형식으로 추가합니다.
- 초기 누적 로그는 [2026-03-09.md](2026-03-09.md)에 보존합니다.
- 작업 중에는 해당 날짜 파일에 변경사항을 한 줄씩 임시 메모합니다.
- 새로운 커밋 묶음을 만들기 직전에는 그 메모를 바탕으로 `무엇 / 왜 / 결과`가 드러나는 요약으로 다시 정리합니다.
- 커밋 후에는 커밋 히스토리와 맞는 요약 형태만 남기고, 중간 메모 성격의 문장은 정리합니다.
