## 1. pnpm Bootstrap — Auto-Install via npm

### What Changed
- `scripts/setup-dev.mjs`: checks `pnpm --version`; if missing, reads `packageManager` from `package.json` and runs `npm install -g pnpm@<version>` automatically
- README Quick Start: first-run command changed from `pnpm setup` → `node scripts/setup-dev.mjs`

### Why It Mattered
Fresh Ubuntu 24 has no pnpm. `pnpm setup` requires pnpm to already exist — circular. `node scripts/setup-dev.mjs` only needs Node (bundled with nvm).

## 2. `pnpm install` Moved Inside setup-dev.mjs

### What Changed
- `pnpm install` runs via `spawnSync` inside `setup-dev.mjs` (after ensure-dev-env, before uv sync)
- Removed `&& pnpm install` suffix from `package.json` `setup` script

### Why It Mattered
Shell `&&` chain is unreliable when `node_modules` is absent; `spawnSync` is unconditional.
