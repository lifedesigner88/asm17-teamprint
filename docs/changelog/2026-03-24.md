## 4. AI Sejong Became A Real Product Feature

### What Changed

- Rebuilt `AI Sejong` from a client-only single-turn box into a login-only multi-turn feature backed by a `persona_chat_messages` table and recent-turn context loading.
- Expanded the grounding so replies now combine persona fields, Creator PR data, and selected Hupository snippets instead of relying on a shallow profile prompt.

### Why It Mattered

- The feature is now easier to operate responsibly because logging, context, and limits are explicit.
- `AI Sejong` behaves much more like a small but real product surface than a demo textbox.
